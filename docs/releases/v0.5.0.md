# Codex-Skin v0.5.0

Codex-Skin v0.5.0 将主题渲染切换为原生 Codex UI 装饰方案，解决自建首页覆盖任务内容、项目输入区和用户图片主体的问题。

## 新功能与修复

- 移除 `#codex-theme-home` 覆盖层，直接装饰 Codex 原生首页、建议卡片、输入框和项目工具栏；
- 使用原生 `home-icon` 路由标记可靠区分首页与任务页；
- 首页显示连续全窗背景，任务页使用更稳健的高可读性表层；
- 根据图片与窗口比例自动选择 `cover` 或 `contain`，兼容用户上传图片；
- 修复 `electron-opaque` 环境中主题背景不可见的问题；
- 稳定切换首页、任务页和侧栏状态，并清理旧版注入节点；
- 强化 macOS 重启与主题清单契约兼容性。

Codex-Skin v0.5.0 replaces the custom home overlay with native Codex UI decoration, preventing themed content from covering tasks, project context, or important parts of user-provided images.

## Highlights

- Decorates the native Codex home, suggestion cards, composer, and project utility bar;
- Reliably separates native home and task routes using the Codex home marker;
- Uses a continuous full-window background on home and a readable surface on tasks;
- Automatically selects `cover` or `contain` based on image and window proportions;
- Fixes background visibility in `electron-opaque` windows;
- Cleans up both legacy overlay injection and new native-route decorations;
- Includes hardened macOS restart behavior and stricter theme manifest compatibility.

> Packages are not commercially code-signed, and macOS PKGs are not notarized. Download only from this Release and verify the SHA-256 checksum before running.
