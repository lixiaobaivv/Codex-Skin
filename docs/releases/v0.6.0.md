# Codex-Skin v0.6.0

Codex-Skin v0.6.0 扩展了主题格式，让主题能够在不同窗口比例下更准确地保护人物和画面主体，并提供安全、声明式的视觉效果。

## 新功能

- 支持 `backgroundFocus` 二维背景焦点，改善宽屏、窄屏下的自动裁切。
- 支持 `visualIntensity` 的清晰、平衡和沉浸三档视觉强度。
- 支持雨、粒子和闪电氛围效果，并遵守系统“减少动态效果”设置。
- 支持任务开始、消息发送时触发的 PNG 叠加素材和输入框装饰。

## 兼容性与安全

- `.dreamskin` 主题包最多支持两张额外特效图片，同时继续兼容旧三文件主题包。
- 主题包上限调整为 28 MiB；特效图片单张上限为 4 MiB、4096×4096。
- 新增严格的路径、尺寸、哈希、签名和触发器校验，主题仍不能携带脚本或自定义 CSS。

## English

Codex-Skin v0.6.0 adds responsive two-dimensional background focus, three visual-intensity presets, safe declarative ambient effects, and optional event-triggered PNG decorations. The package verifier now supports up to two effect assets while preserving compatibility with legacy three-file themes.
