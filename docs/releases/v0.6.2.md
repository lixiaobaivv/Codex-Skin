# Codex-Skin v0.6.2

本版本改进主题应用流程、安全校验和 Windows Codex 安装兼容性。

## 改进

- 将“应用主题”和“应用并重启 Codex”合并为一个自适应主按钮；客户端会检测 Codex 主题端口，只在必要时提示重启。
- 支持更多 Windows Codex 安装形式，包括 Microsoft Store、预览版、企业安装、常见独立安装目录以及 `CODEX_APP_PATH` 自定义目录或可执行文件。
- 启动独立安装版 Codex 时继续将调试端口限制在 `127.0.0.1`。

## 安全

- 严格校验主题字体栈，阻止通过字体字段注入 CSS。
- 公开测试签名密钥仅在测试构建中信任，正式客户端只信任官方主题签名密钥。

## English

Codex-Skin v0.6.2 adds an adaptive apply action, broader Windows Codex installation discovery, safer standalone launch behavior, strict font-stack validation, and production-only trust for official theme signing keys.
