# Codex-Skin v0.4.1

Codex-Skin v0.4.1 修复重启 Codex 后主题首页覆盖空任务或项目页面的问题。

## 修复内容

- 正确识别尚无消息的任务上下文；
- 进入项目、空任务或已有对话时自动隐藏主题首页横幅和快捷卡片；
- 保留真正空白首页的主题展示；
- 增加空任务页面识别的回归测试。

## 下载

- Windows x64：`Codex-Skin-Setup-win-x64.exe`
- macOS Apple Silicon：`Codex-Skin-osx-arm64.pkg`
- macOS Intel：`Codex-Skin-osx-x64.pkg`
- SHA-256：`Codex-Skin-installers-SHA256SUMS.txt`

> 当前安装包未进行商业代码签名，macOS PKG 也未进行 Apple 公证。请仅从本 Release 下载，并在运行前核对 SHA-256。

Codex-Skin v0.4.1 fixes the themed home screen covering empty tasks or project pages after Codex restarts.

## Fixes

- Detect thread context before the first message exists;
- Hide the themed home hero and quick actions in projects, empty tasks, and existing conversations;
- Preserve the themed presentation on the actual blank home screen;
- Add regression coverage for empty-thread detection.

> The packages are not commercially code-signed, and the macOS PKGs are not notarized. Download only from this Release and verify the SHA-256 checksum before running.
