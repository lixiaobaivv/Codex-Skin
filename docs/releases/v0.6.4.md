# Codex-Skin v0.6.4

本版本修复 macOS 首次安装后点击“恢复默认”时出现本机网络请求失败的问题。

- 首次运行且 Codex 尚未开启主题端口时，“恢复默认”会安全判定当前已经是默认主题。
- 不再错误请求未启动的 `127.0.0.1:9229/json/list` 端点。
- 已开启主题模式时仍会正常清理当前主题、注入脚本和视觉节点。

## English

Fixes a macOS first-run error when selecting “Restore Default”. If Codex has not started its local theme endpoint yet, rollback now behaves as a safe no-op instead of reporting a failed request to `127.0.0.1:9229`. Active theme sessions continue to be cleaned normally.
