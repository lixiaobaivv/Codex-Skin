# Codex-Skin v0.6.5

本版本进一步修复 macOS 首次应用主题和恢复默认主题的可靠性问题。

- 首次点击“应用主题”时自动选择热应用或重启 Codex，不再因确认流程异常显示“已取消应用主题”。
- 恢复默认主题遇到本机 CDP 端点瞬时断开时自动重试。
- 所有 `127.0.0.1:9229` 主题请求强制绕过 macOS 系统代理，避免本机请求被代理设置干扰。
- 无主题注入或 Codex 已退出时，恢复默认保持安全且幂等。

## English

Improves first-run theme reliability on macOS. Applying a theme now automatically chooses live injection or a Codex restart without the faulty cancellation path. Local CDP requests bypass system proxies, and restoring defaults retries transient endpoint failures while remaining safe and idempotent when no theme is active.
