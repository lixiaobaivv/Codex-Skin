# Codex-Skin v0.6.3

本版本修复无需重启 Codex 连续切换主题时，侧边栏 Logo 和动态特效节点重复叠加的问题。

- 主题销毁时取消尚未执行的动画帧，防止旧主题在清理后重新插入节点。
- 为主题实例增加销毁状态，已销毁实例不会继续响应 DOM 更新。
- 每次主题更新都会清理失去引用的旧 Logo、氛围层和特效图片。
- 恢复默认主题时也会清扫旧版本遗留的重复节点。

首次应用 `v0.6.3` 中的任意主题即可自动清理当前已经叠加的 Logo，无需手工重启 Codex。

## 在线更新

- 客户端启动后检查官方 GitHub Release，也可以从右上角手动检查。
- 严格筛选 `vX.Y.Z` 稳定客户端版本，不会把 `theme-*` 主题 Release 误识别为应用更新。
- 自动选择当前平台安装包，验证官方 Release 路径、文件大小和 SHA-256 后才允许启动安装。
- 下载过程显示百分比；Windows 静默启动安装程序，macOS 打开对应 PKG。

## English

Fixes duplicated sidebar logos and effect layers after repeatedly switching themes without restarting Codex. Pending animation frames are cancelled, disposed instances are inert, and orphaned nodes from older versions are automatically removed. This release also adds in-app client updates with stable app-release selection, platform-specific assets, progress reporting, size checks, and SHA-256 verification.
