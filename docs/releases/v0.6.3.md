# Codex-Skin v0.6.3

本版本修复无需重启 Codex 连续切换主题时，侧边栏 Logo 和动态特效节点重复叠加的问题。

- 主题销毁时取消尚未执行的动画帧，防止旧主题在清理后重新插入节点。
- 为主题实例增加销毁状态，已销毁实例不会继续响应 DOM 更新。
- 每次主题更新都会清理失去引用的旧 Logo、氛围层和特效图片。
- 恢复默认主题时也会清扫旧版本遗留的重复节点。

首次应用 `v0.6.3` 中的任意主题即可自动清理当前已经叠加的 Logo，无需手工重启 Codex。

## English

Fixes duplicated sidebar logos and effect layers after repeatedly switching themes without restarting Codex. Pending animation frames are cancelled, disposed instances are inert, and orphaned nodes from older versions are automatically removed.
