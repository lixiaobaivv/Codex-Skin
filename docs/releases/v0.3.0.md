# Codex-Skin v0.3.0

Codex-Skin v0.3.0 在增量主题分发基础上加入可见的本地状态、订阅和更新流程。

## 新功能

- 主题卡片显示在线、已下载、已订阅和有更新状态；
- 支持只下载主题资源而不立即应用；
- 支持订阅与取消订阅主题；
- 订阅时自动下载当前版本；
- 刷新远程目录时自动更新全部已订阅主题；
- 持久保存下载版本与订阅状态，客户端重启后继续生效；
- 应用或更新主题后立即刷新界面状态；
- 取消订阅不会删除已经下载的本地主题。

主题资源仍然使用轻量目录、ETag 条件同步、SHA-256 校验、图片解码、原子写入和多线路回退。`.dreamskin` 的 Ed25519/RFC8785 签名导入保持兼容。

## 下载

- Windows x64：`Codex-Skin-Setup-win-x64.exe`
- macOS Apple Silicon：`Codex-Skin-osx-arm64.pkg`
- macOS Intel：`Codex-Skin-osx-x64.pkg`

Codex-Skin v0.3.0 adds visible local state, subscriptions, and updates on top of incremental theme delivery.

## Highlights

- Online, Downloaded, Subscribed, and Update Available states on theme cards;
- Download themes without applying them;
- Subscribe and unsubscribe from themes;
- Download the current version immediately when subscribing;
- Automatically update subscribed themes whenever the remote catalog is refreshed;
- Persist downloaded versions and subscriptions across application restarts;
- Refresh UI state immediately after downloading, updating, or applying a theme;
- Keep local theme data when unsubscribing.

Theme resources continue to use the lightweight catalog, ETag synchronization, SHA-256 validation, image decoding, atomic writes, and transport fallback. Existing Ed25519/RFC8785 `.dreamskin` imports remain compatible.

> The packages are not commercially code-signed, and the macOS PKGs are not notarized. Download only from this Release and verify the SHA-256 checksum before running.
