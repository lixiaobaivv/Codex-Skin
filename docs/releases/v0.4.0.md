# Codex-Skin v0.4.0

Codex-Skin v0.4.0 改进主题在不同窗口尺寸和明暗界面下的显示，并补充本地主题管理能力。

## 新功能与改进

- 支持删除已经下载或安装的主题，并同步清理本地主题状态；
- 优化主题背景图的自适应裁切与定位，减少人物主体和关键画面被界面遮挡；
- 改进深色主题下的对话框、已编辑文件区域和其他界面表面；
- 完善主题 UI 在不同窗口尺寸下的响应式布局；
- 修复官方主题验证器的跨平台构建目标和 Linux 发布依赖；
- 保持主题订阅、更新、签名导入和在线目录兼容。

## 下载

- Windows x64：`Codex-Skin-Setup-win-x64.exe`
- macOS Apple Silicon：`Codex-Skin-osx-arm64.pkg`
- macOS Intel：`Codex-Skin-osx-x64.pkg`
- SHA-256：`Codex-Skin-installers-SHA256SUMS.txt`

> 当前安装包未进行商业代码签名，macOS PKG 也未进行 Apple 公证。请仅从本 Release 下载，并在运行前核对 SHA-256。

Codex-Skin v0.4.0 improves theme rendering across window sizes and light/dark surfaces, and adds local theme removal.

## Highlights

- Delete downloaded or installed themes and clean up their local state;
- Improve adaptive background cropping and positioning to keep important subjects visible;
- Refine dialogs, edited-file surfaces, and other UI areas in dark themes;
- Improve responsive theme layouts across window sizes;
- Fix cross-platform verifier targets and Linux dependencies for official theme publishing;
- Preserve compatibility with subscriptions, updates, signed imports, and the online catalog.

> The packages are not commercially code-signed, and the macOS PKGs are not notarized. Download only from this Release and verify the SHA-256 checksum before running.
