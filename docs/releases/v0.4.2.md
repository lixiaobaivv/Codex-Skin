# Codex-Skin v0.4.2

Codex-Skin v0.4.2 补充修复选择项目但尚未创建任务时，主题首页覆盖项目输入区的问题。

## 修复内容

- 识别输入框中的项目上下文区域；
- 选择项目后立即隐藏主题首页横幅和快捷卡片；
- 同时兼容空任务、已有对话和真正的全局空白首页；
- 增加项目上下文与空任务上下文的回归测试。

## 下载

- Windows x64：`Codex-Skin-Setup-win-x64.exe`
- macOS Apple Silicon：`Codex-Skin-osx-arm64.pkg`
- macOS Intel：`Codex-Skin-osx-x64.pkg`
- SHA-256：`Codex-Skin-installers-SHA256SUMS.txt`

> 当前安装包未进行商业代码签名，macOS PKG 也未进行 Apple 公证。请仅从本 Release 下载，并在运行前核对 SHA-256。

Codex-Skin v0.4.2 fixes the themed home screen covering the composer after a project is selected but before a task is created.

## Fixes

- Detect project context rendered outside the composer surface;
- Hide the themed hero and quick actions immediately after selecting a project;
- Preserve behavior for empty tasks, existing conversations, and the actual global home screen;
- Add regression coverage for project and empty-thread contexts.

> The packages are not commercially code-signed, and the macOS PKGs are not notarized. Download only from this Release and verify the SHA-256 checksum before running.
