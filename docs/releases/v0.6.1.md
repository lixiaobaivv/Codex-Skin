# Codex-Skin v0.6.1

本版本修复企业内网或系统代理环境下，从主题商店一键导入 GitHub Release 主题时等待很久并最终出现 `DSI_NETWORK_BLOCKED` 的问题。

- GitHub 官方发布域名现在可以使用操作系统代理链路。
- 非 GitHub 域名继续使用公网 IP 校验与 DNS 固定，SSRF 防护没有放宽。
- 连接超时缩短为 8 秒、单线路总超时缩短为 25 秒，故障线路能够更快切换。

## English

Fixes slow or blocked one-click imports on corporate networks and system proxies. Exact GitHub release hosts can now use the operating-system proxy, while arbitrary hosts remain protected by public-IP validation and DNS pinning.
